> A TypeScript movie discovery app with fast search, watchlists, and streaming-style UI.

## Overview

Movix is a polished movie browsing experience built for quick discovery. It focuses on fast search, clean entertainment UI patterns, watchlist-style flows, and a frontend that feels familiar to streaming users.

## Quick Start

```powershell
git clone https://github.com/akyourowngames/Movix.git
cd Movix
npm install
```

## Run Locally

```powershell
npm run dev
Open http://localhost:3000
```
